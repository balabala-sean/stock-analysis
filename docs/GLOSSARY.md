# 概念解释

本文档解释项目中涉及的核心概念和术语。

## 核心概念

| 名称             | 说明                                                                                                                                                                                                                                                              |
|----------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **K线数据源**      | 通达信：它是国内最主流的股票交易行情软件，量化社区比较活跃，提供实时行情（A股/港股/期货）、历史K线、技术指标分析、交易下单功能，本项目使用其免费的数据接口来获取股票行情。(相较akshare、yfinance等数据源，通达信在数据质量/网络传输等方面都更加稳定可靠，可以达到商用的程度。如果要做日内高频，对时效性能要求特别高的场景，也可以考虑QMT/JoinQuant等付费解决方案）                                                              |
| **mootdx**     | 本项目核心依赖的Python 第三方库，封装通达信数据协议。支持获取实时行情、历史K线、指数数据，覆盖标准市场（股票）和扩展市场（期货）。                                                                                                                                                                                           |
| **K线 (OHLCV)** | 股票价格的可视化表示方式，每根K线包含：Open（开盘价）、High（最高价）、Low（最低价）、Close（收盘价）、Volume（成交量）。周期可设为1分钟/5分钟/日线/周线等。                                                                                                                                                                    |
| **买点策略**       | 买点信号策略，通过两个维度判断买入时机：(1)价格的均值回归来计算顶底摆动区间；(2)慢速趋势指标增强判断价格的转折点；(1)、(2)同时满足条件时产生买入信号。 均值回归是目前A股市场一种主流的买点策略（策略的实现方案上大同小异，只是策略内部使用的因子不同）                                                                                                                              |
| **卖点策略**       | 本项目未提供卖点策略，卖点在量化交易领域存在非一致性的特征（业内难题），有较多的个性化因素。建议：入门的卖点策略可以尝试均值回归进行实现（例如归一化的值大于90，能达到实盘够用的程度，非最佳的卖点策略，存在优化空间）。                                                                                                                                                   |
| **信号**         | 策略计算产生的结果，具备"点特征"，即某个价格点+时间点的组合，代表一个入场机会（买点信号）或出场机会（卖点信号）。信号是择时策略的核心输出，帮助交易者在合适的时机执行交易决策。                                                                                                                                                                       |
| **指标**         | 对原始数据（价格、成交量等）进行数学变换后得到的计算结果，用于描述市场的某种特征或状态。技术分析中常见的指标如RSI、MACD、KDJ等。指标可以是单个数值，也可以是多个相关数值的组合（复合指标）。本项目中的`PriceMeanIndicator`和`SlowTrendLineIndicator`属于复合指标，内部计算并输出多个因子。                                                                                        |
| **因子**         | 因子是量化领域的核心概念，指能够解释或预测资产收益的单一数值特征。每个因子描述一个独立维度，如动量因子、估值因子、波动率因子等。因子与指标的关系：指标是计算模块，因子是指标的输出结果。例如本项目中的`UP_LINE`、`price_mean_signal`都是单个因子。多因子策略通过组合多个因子来提升信号的稳健性。                                                                                                  |
| **邮件通知**       | 当识别到最新买点时，自动发送邮件通知用户，方便及时跟踪买入信号。需配置 config.json 文件。                                                                                                                                                                                                             |
| **股池**         | 股票池配置，在 config.json 中定义待监控的股票列表，支持批量分析多只股票。  国内主流的量化交易模式，会拆分2个不同的阶段：择股和择时，择股优先，择时次之。举个例子：择股的意思是我在决策要买茅台，还是买英伟达，不同的时代背景/政策背景下，择股会对交易产生较大的影响，择时的理解：我已经决策出要买英伟达，但是要找一个相对合适的价格入场。股池的建立：目前市面主流的做法是依赖主观进行圈选，包括基本面，主营，趋势，换手率等多维度的分析（择股），股池确认后再放入config.json进行实时的监控（择时） |

## 信号输出类型

批量分析完成后，会输出两种类型的信号：

| 信号类型       | 说明                              |
|------------|---------------------------------|
| **买点扫描信号** | 当前触发买入信号，满足策略条件，建议进行复核后，再进行买入配置 |
| **低位扫描信号** | 价格处于低位区间，未来短时间内可能产生买点信号，建议提前关注  |
